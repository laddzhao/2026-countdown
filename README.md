# 2026年春节倒计时网站

一个美观、可更换样式的2026年春节倒计时网站，带有福字烟花效果和多种主题切换功能。

![春节倒计时预览](https://img.shields.io/badge/春节-倒计时-red) ![版本](https://img.shields.io/badge/版本-1.0.0-brightgreen) ![许可证](https://img.shields.io/badge/许可证-MIT-blue) ![GitHub仓库](https://img.shields.io/badge/仓库-2026--countdown-blue)

## 功能特性

- 🎉 **实时倒计时**：精确显示距离2026年春节（2026年2月17日）的剩余时间
- 🎨 **主题切换**：5种不同风格的春节主题（传统红火、现代金耀、水墨丹青、冰雪蓝调、新春绿意）
- ✨ **福字烟花效果**：动态福字烟花从底部升起，营造节日氛围
- 📱 **响应式设计**：适配各种设备屏幕尺寸
- 💾 **主题记忆**：自动保存用户选择的主题
- 🎊 **节日装饰**：包含福字、春联、云朵等春节装饰元素

## 使用方法

1. 下载或克隆本项目到本地
2. 直接在浏览器中打开 `index.html` 文件
3. 或上传到支持静态网站的服务器/托管平台

### 本地运行

```
克隆仓库
git clone https://github.com/zlyup/2026-countdown.git
进入项目目录
cd 2026-countdown
直接在浏览器中打开index.html
```

### GitHub Pages 部署
1. 将项目上传到GitHub仓库（名为2026-countdown）
2. 进入仓库设置 (Settings)
3. 找到 "Pages" 选项
4. 选择主分支 (main/master) 作为源
5. 保存设置，等待部署完成
6. 访问 https://zlyup.github.io/2026-countdown/

## 技术栈
- HTML5
- CSS3 (CSS变量、Flexbox、Grid、动画)
- JavaScript (ES6+)
- Font Awesome 图标库
- 响应式设计

### 主要功能说明
- 倒计时功能
- 实时计算距离2026年2月17日春节的剩余时间
- 精确到秒的自动更新
- 天、小时、分钟、秒分别显示

### 主题切换
- 传统红火：经典春节红色主题
- 现代金耀：金色与黑色结合的现代风格
- 水墨丹青：传统水墨画风格
- 冰雪蓝调：蓝色系冰雪主题
- 新春绿意：绿色系新春主题

### 福字烟花效果
- 多个"福"字从页面底部升起
- 上升过程中逐渐放大、旋转并消失
- 随机位置、大小和颜色变化
- 循环播放，营造持续的节日氛围

- Edge 79+
- Opera 50+
